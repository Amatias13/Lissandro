Como configurar os scripts .bat

1

Descarrega os ficheiros start.bat e stop.bat abaixo e coloca-os na pasta D:\Projetos\Lissandro\

2

Edita o start.bat e confirma que os caminhos do Python e do ngrok estão corretos para o teu sistema.

3

Faz duplo clique em start.bat — abre duas janelas: uma para o bot e outra para o ngrok. O URL novo do ngrok aparece na janela do ngrok.

4

Atualiza o URL na dashboard (Tab Definições → URL da API) sempre que o ngrok mudar (no plano gratuito muda cada vez que reinicias).

5

Para evitar o URL mudar, considera o plano pago do ngrok ou usa um domínio estático gratuito: ngrok config add-authtoken SEU_TOKEN e depois cria um domínio estático no painel do ngrok.

conteúdo dos scripts

start.bat

@echo off
title Lissandro Music Bot
cd /d D:\Projetos\Lissandro

REM Inicia o ngrok numa janela separada
start "ngrok" cmd /k "ngrok http 5555"

REM Espera 2 segundos para o ngrok arrancar
timeout /t 2 /nobreak > nul

REM Ativa o venv e inicia o bot
call .venv\Scripts\activate
python bot.py
pause

stop.bat

@echo off
echo A parar o bot e o ngrok...
taskkill /F /IM ngrok.exe /T 2>nul
taskkill /F /FI "WINDOWTITLE eq Lissandro Music Bot" /T 2>nul
echo Parado.
pause
